# Diplom_3 — Автотесты для UI Stellar Burgers

## Описание проекта
Проект содержит набор автотестов для веб‑приложения **Stellar Burgers**  
https://stellarburgers.education-services.ru  

Тесты написаны на **Python + Pytest + Selenium** с использованием паттерна **Page Object**.  
Тесты покрывают основную функциональность сайта: навигацию, работу с ингредиентами, оформление заказа и проверку счетчиков в ленте заказов.  
Тесты запускаются в **Google Chrome** и **Mozilla Firefox**.  
Поддерживается генерация отчетов через **Allure**.

## Структура проекта

Diplom_3/
│
├── pages/                        # Page Object классы (описание элементов и действий)
│   ├── base_page.py              # Базовый класс: общие методы (клик, ввод текста, ожидания)
│   ├── ingredient_page.py        # Страница ингредиента: модальное окно и его закрытие
│   ├── login_page.py             # Страница логина: поля email/пароль, кнопка "Войти"
│   ├── main_page.py              # Главная страница: переходы, ингредиенты, конструктор, заказ
│   └── order_feed_page.py        # Лента заказов: счетчики и список заказов "В работе"
│
├── tests/                        # Тесты для проверки функциональности
│   ├── test_ingredient.py        # Проверка открытия/закрытия модалки и счетчика ингредиента
│   ├── test_navigation.py        # Проверка переходов: "Конструктор" и "Лента заказов"
│   ├── test_order_feed.py        # Проверка увеличения счетчиков заказов
│   └── test_order_form.py        # Проверка оформления заказа и появления номера в "В работе"
│
├── conftest.py                   # Фикстура для запуска браузеров (Chrome, Firefox)
├── requirements.txt              # Список зависимостей проекта (pytest, selenium, allure и др.)
└── README.md                     # Описание проекта, инструкция по запуску
